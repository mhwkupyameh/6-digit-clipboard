# 📋 6-Digit Online Clipboard

Upload files → Get 6-digit code → Share anywhere!

## ✨ Features
- **6-digit codes** (000000-999999) - super easy to share
- Drag & drop files (images, text, PDFs up to 5MB)
- Live previews for images/text
- Instant download
- Auto-expires after 7 days
- Works offline after first load

## 🚀 Demo
1. Upload any file
2. Get 6-digit code like `847392`
3. Share the code!
4. Anyone enters code → instant download

## 📱 Usage
- Save as `index.html`
- Open in any browser
- No server required!
